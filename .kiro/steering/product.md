# Product Overview

Event Management System (행사 관리 웹서비스) is a web application for managing recurring monthly events. It enables event organizers to send invitation emails, track attendance, and manage participant lists efficiently.

## Core Features

- Google account-based authentication
- Event creation and management with details (name, location, description, instructor, date)
- Multiple attendee registration methods (single email, Excel upload, Google Sheets import)
- Automated invitation emails with Google Forms for attendance confirmation
- Real-time attendance tracking
- Event copying for recurring monthly events
- Google Sheets as the primary database

## Target Users

Event organizers who manage recurring monthly events and need to track participant attendance efficiently.

## Language

The application is designed for Korean-speaking users. UI text, email templates, and user-facing content should be in Korean.
